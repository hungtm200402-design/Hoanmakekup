# Frontend Context

The frontend is the customer-facing web experience for Hoanmakekup.

## Purpose

Customers use the frontend to discover makeup services and cosmetic products, book makeup appointments, and place orders.

## Core Concepts

- **Customer**: A person browsing services/products, booking appointments, or placing orders.
- **Makeup service**: A bookable service offered by the business.
- **Product**: A cosmetic product available for sale or display.
- **Appointment**: A scheduled makeup booking requested by a customer.
- **Order**: A purchase flow for products and possibly service-related items.

## Notes For Agents

- Preserve the distinction between booking workflows and product purchase workflows.
- Use customer-facing language for UI and issue descriptions in this context.
- When a frontend change depends on admin data management, also read `docs/context/backend.md`.
